# 1. Welcome to Music Story app #

You can use app to create, read, update and delete your stories about music and more.
You must have an google e-mail to use this app

# 2. How to install #

Use command (bash) terminal "git clone https://gitlab.com/shopa.tomek/learn-mongodb" in chosen directory on your hard disc.

To install dependencies use command: "npm install (name of dependency)". For example: "npm install connect-mongo"

# 3. How to run #

After all dependencies being installed you can launch app on the local server in your browser
 
 Use "npm run dev" command in terminal to start local server. 
 
 1. Type in the url field in your browser: "http://localhost:5000" to launch app
 2. You can use link from terminal "http://localhost:5000". Press ctrl and click. It will automatically open your brower with app
